# VC VAD モデル v5→v6 移行設計 (2026-07-08)

## 問題

VAD ノイズゲートは Silero VAD v5.1.2 で稼働している。上流は v6 系（最新 v6.2.1）に更新されており、非音声判定の向上が見込める。初期実装時の「後継モデルは入力契約を壊すから使うな」という注記は未検証の防御的仮定で、実アーティファクトを検査した結果、入力契約は一致し判定も向上していることが判明した。

## ゴール

- VAD モデルを v6.2.1 に差し替え、非音声（ブレス・環境音）判定を向上させる。
- ゲートの挙動・既定閾値は変えない（差し替えと表記更新に収束させる）。

## 非ゴール

- ゲートアルゴリズム（スキップ判定・出力ダック・窓解像度マスク）の変更。
- 既定閾値の再チューニング（実機で問題が出たときのみ別途）。
- 定量 A/B ハーネスの構築、v5/v6 のランタイム切替、8kHz 経路。
- 歴史的記録である v5 の spec/plan の改変。

## 受入基準

- [ ] 差し替えるモデルが v6.2.1（sha256 照合済み）である。
- [ ] ゲート有効時の実モデルテストが v6.2.1 で通過する。
- [ ] フルゲート（整形・lint・型・テスト）が green。
- [ ] 実機で、ブレス・環境音のスキップと発話の無傷保持が v5 と同等以上である。
- [ ] ゲート無効（既定）では従来と完全に同一の出力になる。
- [ ] ロールバックは退避した旧モデルを元パスに戻すだけで完了する（コード変更不要）。
